Welcome to the Strongtalk System!

This is the Strongtalk start page. It is designed to be viewed using the Strongtalk browser, not a regular html browser, because it contains Smalltalk code. Pressing the button on the Strongtalk Launcher (which is probably right behind this window) will always open this page.

Feel free to modify this page to contain links to your commonly used commands, classes, and applications. This is useful because the Strongtalk browser, while very primitive in most respects, contains enhancements that let you attach 'do it' code to links (Smalltalk code that is executed when you click on the link). You can also easily imbed Smalltalk applications directly into pages, using the smappl tag. This page contains some examples of these techniques that you can copy and modify. To edit this page, press the button on the toolbar.
